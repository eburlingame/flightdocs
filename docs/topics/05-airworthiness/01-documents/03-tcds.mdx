# Type Certificate Data Sheets

The Type Certificate Data Sheet (TCDS) is a formal description of the aircraft,
engine or propeller published when the aircraft was certified. It lists
limitations and information required for type certification including airspeed
limits, weight limits, thrust limitations, etc.

TCDS are published by the FAA and can be
[found here](https://rgl.faa.gov/Regulatory_and_Guidance_Library/rgMakeModel.nsf/Frameset?OpenPage).
