# Automated Terminal Information Service (ATIS)

An ATIS broadcast is a continuously repeating recording of non-control
information available at larger airport.

The ATIS broadcast typically includes:

- Time of observation
- Latest weather observation
- Cloud cover or vertical obscuration
- Wind
- Altimeter setting
- Temperature / dewpoint
- Active runways
- Pertinent NOTAMs or advisories
