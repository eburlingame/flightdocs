# Type Certificate Data Sheets

The Type Certificate Data Sheet (TCDS) is a formal description of the aircraft, engine or propeller published when the aircraft
was certified. It lists limitations and information required for type certification including airspeed limits, weight limits,
thrust limitations, etc.
